# Generated architecture diagrams

These diagrams are regenerated from the codebase by guardrail tests in `petclinic-backend/src/test/java/.../guardrail/`. Only the source files (`.puml`, `.dsl`) are committed; rendered images are served live by the [PlantUML web service](https://plantuml.com/) from the GitHub-hosted source.

If a renderer ever blocks the URL, swap `www.plantuml.com/plantuml/proxy?...&src=` for `kroki.io/plantuml/svg/<base64>` or render locally with the PlantUML CLI.
