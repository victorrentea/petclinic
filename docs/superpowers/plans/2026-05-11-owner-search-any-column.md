# Owner Search Across Any Column — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the owners list's last-name-only filter with a single search box that matches any column (firstName, lastName, address, city, telephone), substring, case-insensitive, filtered at the DB.

**Architecture:** A new JPQL `@Query` on `OwnerRepository.search(String q)` performs `LOWER(col) LIKE LOWER('%q%')` across the five string columns OR'd together. The controller renames the `lastName` query param to `search`. The Angular owner-list component renames its bound field and updates the input label. OpenAPI and TS types regenerate via the existing pre-commit hook.

**Tech Stack:** Spring Boot 3.5 + Spring Data JPA + JPQL, springdoc-openapi, Angular 16, openapi-typescript.

package victor.training.petclinic.repository;

import java.util.List;
import java.util.Optional;

import org.springframework.data.jpa.repository.Query;
import org.springframework.data.repository.Repository;
import org.springframework.data.repository.query.Param;
import victor.training.petclinic.model.Owner;

public interface OwnerRepository extends Repository<Owner, Integer> {

    @Query("""
        select o from Owner o
        where lower(o.firstName) like lower(concat('%', :q, '%'))
           or lower(o.lastName)  like lower(concat('%', :q, '%'))
           or lower(o.address)   like lower(concat('%', :q, '%'))
           or lower(o.city)      like lower(concat('%', :q, '%'))
           or lower(o.telephone) like lower(concat('%', :q, '%'))
        """)
    List<Owner> search(@Param("q") String q);

    Optional<Owner> findById(int id);

    Owner save(Owner owner);

    List<Owner> findAll();

    void delete(Owner owner);
}
```

@Operation(operationId = "listOwners", summary = "List owners")
@GetMapping(produces = "application/json")
public List<OwnerDto> listOwners(@RequestParam(name = "search", required = false) String search) {
    List<Owner> owners;
    if (search != null && !search.isBlank()) {
        owners = ownerRepository.search(search);
    } else {
        owners = ownerRepository.findAll();
    }
    return ownerMapper.toOwnerDtoCollection(owners);
}
```

searchOwners(search: string): Observable<Owner[]> {
  let url = this.entityUrl;
  if (search !== undefined && search !== '') {
    url += '?search=' + encodeURIComponent(search);
  }
  return this.http
    .get<Owner[]>(url)
    .pipe(catchError(this.handlerError('searchOwners', [])));
}

- [ ] **Step 2: Update the component class**

In `owner-list.component.ts`, replace the body with:

```typescript
import {Component, OnInit} from '@angular/core';
import {OwnerService} from '../owner.service';
import {Owner} from '../owner';
import {Router} from '@angular/router';
import { finalize } from 'rxjs/operators';

@Component({
  selector: 'app-owner-list',
  templateUrl: './owner-list.component.html',
  styleUrls: ['./owner-list.component.css']
})
export class OwnerListComponent implements OnInit {
  errorMessage: string;
  search: string;
  owners: Owner[];
  isOwnersDataReceived: boolean = false;

  constructor(private router: Router, private ownerService: OwnerService) {
  }

  ngOnInit() {
    this.ownerService.getOwners().pipe(
      finalize(() => {
        this.isOwnersDataReceived = true;
      })
    ).subscribe(
      owners => this.owners = owners,
      error => this.errorMessage = error as any);
  }

  onSelect(owner: Owner) {
    this.router.navigate(['/owners', owner.id]);
  }

  addOwner() {
    this.router.navigate(['/owners/add']);
  }

  runSearch(search: string) {
    if (search === '' || search === undefined) {
      this.ownerService.getOwners().subscribe(owners => this.owners = owners);
      return;
    }
    this.ownerService.searchOwners(search).subscribe(
      owners => this.owners = owners,
      () => this.owners = null
    );
  }
}
```
